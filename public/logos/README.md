# 브랜드 로고 (경쟁사 대시보드용)

이 폴더에 브랜드 로고 PNG를 넣으면 대시보드에서 `/logos/<파일명>` 으로 참조됩니다.
- 형식: 투명 배경 PNG, 높이 40px 내외 권장
- 파일명: 소문자 브랜드키

## 필요 파일
- lg.png
- samsung.png
- sony.png
- tcl.png
- hisense.png
- midea.png

추가 브랜드는 `소문자브랜드명.png`로 추가 (예: panasonic.png, sharp.png, tcl.png).
매핑에 없는 브랜드는 자동으로 브랜드 컬러 워드마크로 폴백됩니다.
